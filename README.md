# OLX UI Automation Lab 2

##  Мета роботи
Отримання практичних навичок з автоматизації UI-тестів за допомогою Selenium WebDriver та реалізація патерну Page Object Model (POM).

##  Налаштування середовища
* **Мова:** JavaScript (Node.js).
* **Бібліотеки:** Selenium WebDriver, Mocha, Assert.
* **Браузер:** Google Chrome.

##  Архітектура
Проєкт побудований за принципом розділення логіки сторінок та самих тестів:
* `/pages` - класи-описи сторінок (локатори та методи):
    * `MainPage.js` - головна сторінка.
    * `LoginPage.js` - форма авторизації.
    * `SearchResultsPage.js` - видача оголошень та фільтри.
* `/tests` - файл `manual.test.js` із 7 сценаріями.

##  Опис автоматизованих тестів
У межах роботи автоматизовано такі сценарії:
1. **TC-01:** Пошук товару за ключовим словом "iPhone 15".
2. **TC-02:** Фільтрація оголошень за діапазоном ціни (10000 - 20000 грн).
3. **TC-03:** Сортування результатів за датою публікації.
4. **TC-04:** Перевірка негативного сценарію авторизації (невірний пароль).
5. **TC-05:** Додавання товару в "Обране" неавторизованим користувачем (перевірка виклику модального вікна).
6. **TC-08:** Навігація по головним категоріям (Транспорт).
7. **TC-09:** Застосування фільтра "Тільки з фото".

##  Запуск проєкту
1. Схилити репозиторій.
2. Встановити залежності: `npm install`.
3. Запустити тести: `npm test`.
